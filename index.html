Capabilities

What it does

One toolkit for the full Magento 2 lifecycle — read-only until you approve a change.

Scaffold

New modules, EAV attributes, GraphQL/REST APIs, admin forms & grids, CLI commands, and message queues.

Review & Audit

Severity-ranked findings with file:line evidence, plus one consolidated release-readiness audit with merged SARIF.

Test

Generate unit, integration, REST/GraphQL API, JavaScript, and MFTF tests with real assertions.

Fix & Debug

Reproduce → root-cause analysis → minimal TDD fix → regression test, with a DI/log investigator when repro is hard.

Deploy & Release

Pre-flight validation, ordered deploy with rollback, and versioned releases with generated notes.

Context-aware

Hub-and-spoke around magento2-context — adapts to Docker or bare PHP, Luma, Hyvä, or Breeze, any vendor.
