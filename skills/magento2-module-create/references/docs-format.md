# Documentation Format Rules

This skill does not define its own `README.md` or `CHANGELOG.md` format — both are delegated in full
to `magento2-docs-generate` (Step 6), which owns the single canonical template for each: the README
section structure lives in `magento2-docs-generate/templates/readme.md` and
`magento2-docs-generate/references/doc-structure.md`; the CHANGELOG structure and category vocabulary
follow the canonical `magento2-context/references/changelog-format.md`. See
`references/documentation-guide.md` for how and when the delegation runs.
